Read me MACOS : 

-Télécharger le fichier Gamma3D_MacOS.zip

-Une fois le fichier Gamma3D_MacOS.zip téléchargé cliquer sur Gamma_Instal.dmg

-Une fois Gamma_Instal.dmg exécuté aller dans le disque Gamma_Instal

-glisser le dossier "laby" et l'application adaptée au Mac dans le dossier Application (pour Intel Gamma3D_Intel.app, pour AppleSilicone Gamma3D_Silicone.app)

-Exécuter l'application dans le Dossier Application


NOTE IMPORTANTE : le fichier Gamma3D_XXX.App a besoin d'être exécuté dans le même dossier que le dossier laby. En toute logique ils sont déjà dans le même dossier si vous avez suivit les instruction de ce ReadMe, mais si l'un vient à être changé d'emplacement il faut aussi changer l'emplacement de l'autre.
