Read me WINDOWS : 


-Télécharger le fichier Gamma3D_Windows.zip

-Une fois le fichier Gamma3D_Windows.zip téléchargé, décompresser l'archive a un emplacement au choix

-Ouvrir le dossier Gamma3D_Windows

-Dans le dossier Gamma3D_Windows exécuter le fichier Gamma3D.exe


NOTE IMPORTANTE : le fichier Gamma3D.exe a besoin d'être exécuté dans le même dossier que le dossier laby. En toute logique ils sont déjà dans le même dossier mais si l'un vient a être changé d'emplacement il faut aussi changer l'emplacement de l'autre.
