Rendu final du projet RayTracer :

Dans ce dossier vous pouvez trouver les sources du projet dans le dossier raytracer-template. 

Le projet a été réalisé sur l'IDE Clion tout au long de l'année, il est fonctionnel sur cet IDE et en ligne de commande.

Les phase 1,2 et 4 on été faites et tout es fonctionnel comme on peut le voir dans l'output nommé Acte_final_Dernière_Scene. Le seul point non abordé a été celui de la phase 3 sur le maillage.
